Class query

Class for holding and handling queries and query-manipulation pre-execution

Author: Thomas Mundal <thmundal@gmail.com>
Located at mysql.php

public
# __construct( array $query_parameters )

Initializes a new query and translates the parameters into usable SQL. Query parameters are passed in an array by identifying information trough key/value pairs. Key/value pairs that can be used are:

"type" => string "select"/"insert"/"delete"/"update",
"table" => table_name,
"data" => array dataset_of_keyvalue_pairs
"where" => (string) condition ex: "row1 = true AND row2 = \"blah\"",
"order" => mysql_order_statement,
"limit" => mysql_limit_statement

Example of an update query:

$connection->execute(new query([
"type" => "update",
"table" => "test",
"data" => ["name" => "update test 2"],
"where" => "id = 1"
]));
